# Slot Machine Game 🎰

A fully functional command-line slot machine game built with Python.
The player deposits money, chooses lines to bet on, places a bet, and spins to win.

## How It Works
1. Deposit an amount to start
2. Choose the number of lines to bet on (1–3)
3. Place your bet per line
4. Spin the slot machine
5. Win or lose based on matching symbols

## Features
- Deposit and balance tracking
- Betting system with min and max limits
- Random slot machine spin logic
- Symbol frequency and value system
- Win detection across multiple lines
- Input validation throughout

## Tech Stack
- Python 3

## Symbols & Values
| Symbol | Count | Value |
|--------|-------|-------|
| A      | 2     | 5x    |
| B      | 4     | 4x    |
| C      | 6     | 3x    |
| D      | 8     | 2x    |

## How to Run
1. Make sure Python is installed
2. Clone the repository or download the file
3. Run the game:
```bash
python betting.py
```
